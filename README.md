# Metal Prices App

##  Description
This is a React Native app that displays live prices of metals like Gold, Silver, Platinum, and Palladium.

##  Features
- Home screen with metal prices
- Independent loading for each metal
- Error handling
- Details screen with more information

##  Tech Used
- React Native (Expo)
- React Navigation

##  How to Run
npm install  
npm start  

##  Project Structure
src/
  screens/
    HomeScreen.js
    DetailsScreen.js
